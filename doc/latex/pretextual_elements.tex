\begin{resumo}
    Este relatório apresenta o desenvolvimento do Vitrine, uma plataforma de gerenciamento de conteúdo (CMS) open source projetada para democratizar a publicação independente. O problema central reside na barreira tecnológica enfrentada por docentes e pesquisadores que, ao buscarem autonomia digital, esbarram na complexidade de infraestruturas tradicionais ou na rigidez de plataformas proprietárias. A pesquisa, de natureza aplicada e caráter exploratório-descritivo, utilizou uma metodologia de desenvolvimento experimental e incremental para a construção de um artefato tecnológico moderno. O sistema foi construido sobre o framework Next.js, integrando o ORM Prisma com banco de dados SQLite para garantir a portabilidade dos dados e simplicidade no deployment. Os resultados alcançados incluem a implementação de um editor Markdown com suporte a fórmulas em LaTeX e um sistema de personalização visual dinâmica via variáveis CSS, refletidas instantaneamente através de Incremental Static Regeneration (ISR). A solução equilibra o alto desempenho de sites estáticos com a flexibilidade de sistemas dinâmicos, oferecendo uma ferramenta extensível que elimina o trabalho repetitivo de configuração para produtores de conteúdo acadêmico.

    \vspace{\onelineskip}
    \noindent
    \textbf{Palavras-chave}: CMS Acadêmico; Next.js; Open Source; Markdown; Iniciação Científica.
\end{resumo}
